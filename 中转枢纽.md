## 六、区域中转策略

### 6.1 华北"减压阀"策略
- **场景**：北京票价过高或售罄
- **替代方案**：
  - 天津滨海机场 + 京津城际（30分钟到北京）
  - 石家庄机场 + 京石高铁

### 6.2 西北"扇形"辐射策略
- **场景**：前往新疆、青海、甘肃
- **路径**：出发地 → 西安/兰州 → 目的地
- **节点优先级**：西安 > 兰州 > 乌鲁木齐

### 6.3 长三角"多点互备"策略
- **场景**：上海机票昂贵
- **替代方案**：
  - 杭州萧山机场 + 高铁至上海
  - 无锡硕放机场 + 高铁至上海
  - 南京禄口机场 + 高铁至上海

### 6.4 国际入境中转策略
- **场景**：国际航班 → 国内小城市
- **推荐枢纽**（按优先级）：
  1. 北京、上海、广州（一线门户）
  2. 成都、深圳、西安（区域门户）
  3. 郑州、武汉、长沙（中部枢纽）

## 七、算法配置建议

### 7.1 枢纽权重等级

```
Level 1 (最高): 北京、上海、广州
Level 2 (高):   深圳、成都、重庆、西安、武汉、郑州
Level 3 (中):   南京、杭州、长沙、昆明、沈阳、哈尔滨
Level 4 (区域): 其他省会城市
```

### 7.2 中转时间预设

| 中转类型 | 最小中转时间 |
|----------|--------------|
| 空铁一体化（第一梯队） | 60-90分钟 |
| 轨道交通连接（第二梯队） | 120分钟 |
| 摆渡车接驳（第三梯队） | 150分钟 |
| 同机场航班中转 | 90-120分钟 |
| 同城跨机场中转 | 240分钟 |
| 同城跨火车站中转 | 90-120分钟 |
| 铁路同站换乘 | 30-60分钟 |

### 7.3 惩罚权重建议

| 场景 | 惩罚系数 |
|------|----------|
| 跨机场中转 | 2.0x |
| 需摆渡车 | 1.3x |
| 普速火车中转 | 1.5x |
| 夜间中转（22:00-06:00） | 1.8x |
